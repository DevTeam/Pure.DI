## План статьи

### Введение в DI
- Что такое Dependency Injection (на примере, аналогии)
- Что такое Pure DI (чистое внедрение зависимостей без контейнеров)
- Понятие графа зависимостей
- Композиция объектов
- Корни композиции

### Что такое Pure.DI
- Основная идея (генератор кода)
- Ключевые возможности:
  - Zero Overhead
  - Compile-Time Validation
  - Works everywhere
  - Transparency
  - Built-in BCL Support
- Достоинства по сравнению с классическими DI контейнерами

### Быстрый старт
